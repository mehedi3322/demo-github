1. var: Old way to declare variables in JavaScript.
let: Modern way to declare variables.
const: Used to declare constant variables
2.The spread operator is used to expand or copy elements of an array
3. map(): Creates a new array by transforming each element.

filter(): Creates a new array with elements that pass a condition.

forEach(): Loops through an array but does not return a new array.
4.An arrow function is a shorter way to write a function in JavaScript.
5.Template literals allow you to insert variables inside strings using backticks .
